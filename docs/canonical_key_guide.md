# Canonical Key Guide

## Regex
All canonical keys must match:
`^[a-z0-9_\/]+$`

## Rules
1. **Lowercase only**: No uppercase letters.
2. **Snake_case**: Use underscores for spaces.
3. **Forward slash**: Use `/` for hierarchy.
4. **No special chars**: No `@`, `#`, `$`, etc.

## Examples (10-15)

1. `user/profile/user_123/favorite_color`
2. `user/profile/user_123/first_name`
3. `user/preferences/user_123/theme_mode`
4. `world/fact/ohms_law/definition`
5. `world/fact/speed_of_light/value_mps`
6. `world/geography/paris/country`
7. `mace/config/router/default_depth`
8. `mace/config/memory/max_retries`
9. `agent/math/status/health`
10. `agent/profile/version/current`
11. `session/context/last_topic`
12. `user/history/user_123/last_login`
13. `system/state/boot_time`
14. `knowledge/science/water/boiling_point_c`
15. `knowledge/history/ww2/start_year`
